# WiCS Website

Static website mockup for Women in Computer Science at ASU.

## Pages

- `index.html` - landing page, hero, community gallery, pillars, sponsors band
- `events.html` - events calendar, event detail panel, past events
- `team.html` - faculty advisors and officer team
- `sponsors.html` - sponsor logo page

## Assets

- `images/branding/` - WiCS logo and hero moth artwork
- `images/home-page-images/hero-photos/` - gallery and event photos
- `images/team/` - advisor and officer images
- `images/sponsors/` - sponsor logos

## Local Preview

Run a simple static server from the project root:

```bash
python3 -m http.server 8085
```

Then open `http://localhost:8085`.
